# 🚀 Guide de Démarrage Rapide - DuDirecte

## 1️⃣ Ouvrir l'Application

1. Ouvrez le dossier `DuDirecte` dans un navigateur
2. Ouvrez `index.html` (double-clic ou glisser-déposer)

## 2️⃣ Se Connecter

**Utiliser les identifiants de démo:**
```
Email: demo@ecole.fr
Mot de passe: password123
```

**Ou créer un compte personnalisé:**
```
Email: votre@email.com
Mot de passe: (min 6 caractères)
```

## 3️⃣ Explorer le Dashboard

Le dashboard affiche:
- 📈 Votre niveau (XP progression)
- 🔥 Votre streak actuelle
- ⏱️ Temps d'écran disponible
- 🏆 Votre ligue
- 📚 Devoirs à faire
- 🏅 Top 5 classement

## 4️⃣ Compléter un Devoir

1. Allez sur **"Leçons"**
2. Cliquez sur **"Marquer comme fait"**
3. +10 XP et +5 min de temps d'écran! 🎉

## 5️⃣ Consulter vos Notes

Allez sur **"Notes"** pour voir:
- Toutes vos notes par matière
- Moyenne par matière (calculée auto)
- XP gagné par note

## 6️⃣ Vérifier votre Classement

Sur **"Ligues":**
- Voyez votre position dans le top 30
- Progression vers promotion
- Système de médailles

## 7️⃣ Personnaliser l'App

Sur **"Paramètres":**
- 🎨 Changez le thème (5 options)
- 🌙 Activez le mode sombre
- 🔔 Configurez les notifications
- ⏰ Réglez l'heure des rappels

## 🎮 Gestion des Points

### Vous Gagnez
| Action | Points |
|--------|--------|
| Devoir complété | +10 XP, +5 min |
| Note reçue | Note × Coefficient = XP |
| Streak atteint 7j | Motivation! 🔥 |

### Vous Perdez
| Action | Points |
|--------|--------|
| Absence injustifiée | -10 min |
| Observation | -5 min |
| Heure de colle | -30 min |
| Exclusion | -60 min |

## 💡 Astuces

1. **Streak**: Complétez vos devoirs chaque jour pour un streak croissant
2. **Ligues**: Top 10 monte, derniers 10 descend chaque semaine
3. **Notifications**: Activez les rappels quotidiens pour ne rien oubli
4. **Thème**: Changez le thème selon votre humeur!
5. **Profil**: Personnalisez votre nom dans les paramètres

## 📱 Fonctionnement Mobile

L'app est totalement responsive:
- Sidebar se cache sur mobile
- Menus adaptatifs
- Toucher-friendly

## 🔒 Sécurité

- ✅ Vos données sont stockées **localement** sur votre appareil
- ✅ Les messages sont **chiffrés de bout en bout**
- ✅ Aucun serveur, aucune fuite possible

## 🐛 Dépannage

**"Je suis bloqué à la connexion"**
→ Utilisez `demo@ecole.fr` / `password123`

**"Mes données ont disparu"**
→ Elles sont dans localStorage. Videz le cache du navigateur avec prudence!

**"Le thème ne change pas"**
→ Rafraîchissez la page (F5)

**"Les notifications ne s'affichent pas"**
→ Vérifiez dans Paramètres que les notifications sont activées
